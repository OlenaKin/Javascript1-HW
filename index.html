JavaScript Usage Examples

Вкажи всі можливі способи коментування коду:

  1. // Цей коментар займає весь рядок alert('Привіт');
  2. alert('Світ'); // Цей коментар міститься після інструкції
  3. * Приклад із двома повідомленнями. Це багаторядковий коментар. */ alert('Привіт'); alert('Світ');
  4. /* Закоментований код alert('Привіт'); */ alert('Світ');

Які стилі написання імен змінних ти знаєш?

  1. Camel Case - для імен змінних, функцій та об'єктів. Приклад: camelCase (друге слово з великої літери)
  2. Pascal Case - для імен класів та конструкторів. Приклад: MyClass (кожне слово з великої літери)
  3. Snake Case - іноді для констан або деяких стилів коду. Приклад: my_variable (слова з маленької літери розділяються нижнім підкресленням)
  4. Kebab Case - в більшості випадків для CSS-класів та імен файлів. Приклад: my-class (слова розділені дефісом)
  5. Upper Snake Case - для констант Приклад: MY_CONSTANT (всі букви великі, розділені нижнім підкресленням)
